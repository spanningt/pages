About the show

At The Routing Loop we focus on all aspects of AWS Networking and cover different types of services and architectures. You'll learn about different AWS networking areas:

  • Connectivity services that help with communication within as well as to and from AWS
  • Application networking focused on upper layers of the OSI model that help with more inteligent traffic routing
  • Network security governing placement of security controlls within your network
  • Observability and monitoirng of network components
  • Automation and operations

    • Our mission is educating and supporting builders through their AWS journey with AWS networking and content delivery.
